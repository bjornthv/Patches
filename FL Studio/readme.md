These files are usually installed in:

C:\users\\%user%\Documents\Image-Line\FL Studio\Presets\Plugin presets\\%pluginname%

If you have your documents and/or presets in a different location, use that location instead.
This is an indication, not a manual.


**NOTE**: The "Projects" folder requires a different location. Please read the Readme in the **Projects** folder for more information.
